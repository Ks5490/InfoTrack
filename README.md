# InfoTrack
Conveyancing Data Gathering App

Please use the docker compose file for the simplest setup 
- simply run `docker compose up` and the UI will be available at http://localhost:5173/

Alternatively, you can run the app without docker using visual studio for the backend and VScode for the frontend
Please note you may need to modify the fetch url in "gather-client-data" to match your VS configuration if the port in launch settings is already in use
